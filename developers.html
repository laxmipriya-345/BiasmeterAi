Check Single Decision
POST /api/bias/check
Analyze a single AI decision for bias in real-time. Perfect for integrating into your decision pipelines.
JavaScript Example 
fetch('https://api.biasmeter.ai/v1/check', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    "decision_id": "dec_001",
    "gender": "female",
    "experience": 5,
    "position": "developer",
    "selected": false,
    "industry": "hiring"
  })
})
.then(response => response.json())
.then(data => console.log(data));
Batch Analysis
POST /api/bias/batch
Upload multiple decisions for comprehensive bias analysis. Returns aggregated fairness metrics.
Python Example 
import requests

response = requests.post(
    'https://api.biasmeter.ai/v1/batch',
    headers={'Authorization': 'Bearer YOUR_API_KEY'},
    json={
        "decisions": [
            {"gender": "male", "selected": True},
            {"gender": "female", "selected": False},
            # ... more decisions
        ],
        "industry": "hiring"
    }
)

print(response.json()['fairness_score'])
